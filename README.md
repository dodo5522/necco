# 昭島地域通貨ねっこの通帳管理サーバアプリケーション

個人的な勉強を目的とした、地域通貨通帳管理サーバアプリケーション構築環境一式。  
バックエンドにPython+Flask+MySQLで、フロントエンドは普通にHTML5+bootstrapを使う予定。
